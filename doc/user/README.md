# 사용자 문서

Jangbogo 사용자 및 배포 담당자를 위한 문서입니다.

---

## 📚 문서 목록

### 1. [빌드 가이드](BUILD_GUIDE.md)
**대상:** 배포 패키지 빌드 담당자

**내용:**
- Custom JRE 생성 (jlink)
- packageDist 태스크 실행
- 빌드 옵션 커스터마이징
- 트러블슈팅

**빌드 명령:**
```bash
.\gradlew clean bootJar createJre packageDist
```

---

### 2. [배포 가이드](DEPLOYMENT_GUIDE.md)
**대상:** 시스템 관리자, 최종 사용자

**내용:**
- ZIP 파일 설치 방법
- Jangbogo.bat 실행
- Windows 서비스 등록
- 관리자 계정 설정
- 문제 해결 (트러블슈팅)

**빠른 시작:**
1. ZIP 압축 해제
2. `Jangbogo.bat` 실행
3. 브라우저 접속

---

### 3. [사용자 매뉴얼](USER_GUIDE.md)
**대상:** 최종 사용자

**내용:**
- 서비스 접속 및 로그인
- 쇼핑몰 계정 연결
- 구매내역 수집 (수동/자동)
- 구매정보 저장 및 내보내기
- FAQ (자주 묻는 질문)

**주요 기능:**
- 🛒 다중 쇼핑몰 지원
- 📊 자동 수집
- 💾 다양한 포맷 지원

---

### 4. [변경 이력](../../CHANGELOG.md)
**대상:** 모든 사용자

**내용:**
- 버전별 새로운 기능
- 개선 사항 및 버그 수정
- 알려진 제한사항
