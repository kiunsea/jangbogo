 1. 서비스 접속 및 로그인
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 [1-1] 서비스 시작하기
 ┌────────────────────────────────────────┐
 │ 1) Jangbogo.bat 파일을 더블클릭       │
 │ 2) 콘솔 창이 열리면서 시작 메시지 표시│
 │ 3) 브라우저가 자동으로 열림            │
 └────────────────────────────────────────┘

 ※ 브라우저가 자동으로 열리지 않는 경우:
    → 수동으로 브라우저를 열고 http://127.0.0.1:8282 입력

 [1-2] 로그인하기
 ┌────────────────────────────────────────┐
 │ 로그인 화면에서:                       │
 │                                        │
 │ • 아이디: admin_main                  │
 │ • 비밀번호: admin1234_main            │
 │                                        │
 │ [로그인] 버튼 클릭                    │
 └────────────────────────────────────────┘

 ✓ 로그인 성공 시 → 장보고 관리화면으로 이동
 ✓ 우측 상단에 사용자 이름 표시

 2. 쇼핑몰 계정 연결하기
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 [2-1] 지원 쇼핑몰
 ┌────────────────────────────────────────┐
 │ • SSG (신세계, 이마트, 트레이더스)     │
 │ • OASIS (오아시스)                     │
 │ • HANARO (하나로마트)                  │
 └────────────────────────────────────────┘

 [2-2] 계정 연결 절차
 ───────────────────────────────────────────

 STEP 1: 쇼핑몰 선택
 ┌────────────────────────────────────────┐
 │ 1) 화면 하단 "쇼핑몰 목록" 섹션 확인  │
 │ 2) 연결하려는 쇼핑몰 찾기             │
 │    예: SSG(신세계)                     │
 └────────────────────────────────────────┘

 STEP 2: 계정연결 버튼 클릭
 ┌────────────────────────────────────────┐
 │ 1) 해당 쇼핑몰의 [계정연결] 버튼 클릭│
 │ 2) 팝업 창이 열림                      │
 └────────────────────────────────────────┘

 STEP 3: 쇼핑몰 아이디/비밀번호 입력
 ┌────────────────────────────────────────┐
 │ [쇼핑몰 계정 연결 팝업]                │
 │                                        │
 │ 아이디: [_____________]               │
 │ 비밀번호: [_____________]             │
 │                                        │
 │     [Close]    [계정연결]             │
 └────────────────────────────────────────┘

 ※ 주의사항:
   - 장보고 서비스 계정이 아닌 쇼핑몰 계정을 입력하세요
   - 예: SSG.com 회원 아이디/비밀번호
   - 입력한 정보는 암호화되어 안전하게 저장됩니다

 STEP 4: 연결 확인
 ┌────────────────────────────────────────┐
 │ ✓ 연결 성공 메시지 표시                │
 │ ✓ [계정연결] 버튼 색상 변경            │
 │ ✓ 이제 해당 쇼핑몰에서 구매내역 수집   │
 │   가능                                 │
 └────────────────────────────────────────┘

 [2-3] 여러 쇼핑몰 연결하기
 ───────────────────────────────────────────
 • 위 절차를 반복하여 모든 쇼핑몰 연결 가능
 • 각 쇼핑몰마다 별도로 계정 연결 필요

 3. 구매내역 수집하기 (수동/자동)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 [3-1] 수동 수집 (즉시 수집)
 ───────────────────────────────────────────

 STEP 1: 수집할 쇼핑몰 선택
 ┌────────────────────────────────────────┐
 │ 1) 쇼핑몰 목록에서 원하는 쇼핑몰의     │
 │    체크박스 선택                       │
 │                                        │
 │    예:                                 │
 │    ☑ SSG(신세계)                      │
 │    ☑ OASIS(오아시스)                  │
 │    ☐ HANARO(하나로마트)               │
 └────────────────────────────────────────┘

 STEP 2: 자동수집 버튼 클릭
 ┌────────────────────────────────────────┐
 │ 1) "쇼핑몰 목록" 우측 상단의          │
 │    [자동수집] 버튼 클릭                │
 │                                        │
 │ 2) 수집 진행 중 메시지 표시            │
 │    "수집 중입니다..."                  │
 │                                        │
 │ 3) 수집 완료 메시지 확인               │
 │    "총 N개 쇼핑몰 수집 완료"           │
 └────────────────────────────────────────┘

 ※ 수집 소요 시간:
   - 쇼핑몰당 약 10~30초 소요 (구매내역 양에 따라 다름)
   - 여러 쇼핑몰 선택 시 순차적으로 수집

 [3-2] 자동 수집 (스케줄 설정)
 ───────────────────────────────────────────

 STEP 1: 자동 수집 주기 설정
 ┌────────────────────────────────────────┐
 │ 1) 쇼핑몰 목록에서 해당 쇼핑몰 찾기   │
 │                                        │
 │ 2) "자동 수집 주기(분)" 입력란에      │
 │    원하는 주기 입력                    │
 │                                        │
 │    예시:                               │
 │    • 30  → 30분마다 자동 수집         │
 │    • 60  → 1시간마다 자동 수집        │
 │    • 1440 → 24시간마다 자동 수집      │
 │    • 0   → 자동 수집 안 함            │
 └────────────────────────────────────────┘

 STEP 2: 체크박스 선택
 ┌────────────────────────────────────────┐
 │ • 주기 입력 후 우측 체크박스 선택     │
 │ • 체크 시 자동 수집 활성화             │
 └────────────────────────────────────────┘

 STEP 3: 자동수집 버튼으로 설정 저장
 ┌────────────────────────────────────────┐
 │ 1) [자동수집] 버튼 클릭                │
 │ 2) "자동 수집 설정이 저장되었습니다"  │
 │    메시지 확인                         │
 │ 3) 설정된 주기마다 자동으로 수집 시작 │
 └────────────────────────────────────────┘

 [3-3] 수집 결과 확인
 ───────────────────────────────────────────
 • 콘솔 창에서 실시간 수집 상황 확인 가능
 • 로그 파일: logs/jangbogo.log
 • 수집된 데이터는 데이터베이스에 자동 저장

 4. 구매정보 저장 옵션 설정
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 화면 중간의 "구매정보 저장 옵션" 섹션에서 설정

 [4-1] 저장 위치 설정
 ───────────────────────────────────────────
 ┌────────────────────────────────────────┐
 │ 저장 위치: [./exports        ] [🏠]   │
 └────────────────────────────────────────┘

 • 기본값: ./exports (프로그램 폴더 내 exports)
 • [🏠] 버튼: Windows 내문서 폴더로 변경
 • 직접 입력: 원하는 폴더 경로 입력 가능
   예: C:\Users\사용자명\Documents\장보고

 ※ 폴더가 없으면 자동으로 생성됩니다.

 [4-2] 저장 포맷 선택
 ───────────────────────────────────────────
 ┌────────────────────────────────────────┐
 │ 저장 포맷: [JSON ▼]                   │
 │                                        │
 │ 선택 가능:                             │
 │ • JSON  - 프로그램 간 데이터 교환      │
 │ • CSV   - 엑셀에서 바로 열기 가능     │
 │ • Excel - 엑셀 파일 형식              │
 └────────────────────────────────────────┘

 [4-3] 자동 저장 설정
 ───────────────────────────────────────────
 ┌────────────────────────────────────────┐
 │ 구매내역 수집시 함께 저장: ☐          │
 └────────────────────────────────────────┘

 • ☑ 체크: 구매내역 수집 완료 시 자동으로 파일 저장
 • ☐ 해제: 수동으로만 파일 저장

 ※ 자동 저장 장점:
   - 수집할 때마다 최신 데이터 자동 백업
   - 별도로 저장 버튼을 누를 필요 없음

 [4-4] 설정 저장
 ───────────────────────────────────────────
 • 설정 변경 후 자동으로 저장됨
 • 브라우저를 닫아도 설정 유지

 ┌────────────────────────────────────────┐

 ───────────────────────────────────────────

 [5-3] 저장된 파일 확인
 ───────────────────────────────────────────

 파일 이름 형식:
 ┌────────────────────────────────────────┐
 │ jangbogo_YYYYMMDD_HHMMSS.확장자       │
 │                                        │
 │ 예시:                                  │
 │ • jangbogo_20251104_153045.json       │
 │ • jangbogo_20251104_153045.csv        │
 │ • jangbogo_20251104_153045.xlsx       │
 └────────────────────────────────────────┘

 [5-4] 저장된 데이터 내용
 ───────────────────────────────────────────

 각 구매내역마다 다음 정보 포함:
 ┌────────────────────────────────────────┐
 │ • 구매 일시 (날짜/시간)                │
 │ • 쇼핑몰 이름 (SSG, OASIS 등)         │
 │ • 매장명 (구체적인 매장 위치)          │
 │ • 주문번호 또는 영수증 바코드          │
 │ • 구매 상품 목록                       │
 │   - 상품명                             │
 │   - 수량                               │
 │   - 금액                               │
 └────────────────────────────────────────┘

 [5-5] 저장 파일 활용 예시
 ───────────────────────────────────────────

 JSON 파일:
 • 다른 프로그램에서 데이터 분석
 • 백업 및 데이터 이관

 CSV 파일:
 • 엑셀에서 바로 열기
 • 표로 정리하여 확인
 • 월별/연도별 구매 통계 작성

 Excel 파일:
 • 엑셀에서 바로 열기
 • 서식이 적용된 상태로 확인

 6. 자주 묻는 질문 (FAQ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

 Q1: 구매내역이 수집되지 않아요
 ───────────────────────────────────────────
 A: 다음을 확인해주세요:
    1) 쇼핑몰 계정이 올바르게 연결되었는지 확인
    2) 쇼핑몰 웹사이트에서 정상 로그인 가능한지 확인
    3) 최근 6개월 이내 구매내역이 있는지 확인
    4) 콘솔 창 또는 logs/jangbogo.log 에서 오류 확인

 Q2: 저장된 파일을 찾을 수 없어요
 ───────────────────────────────────────────
 A: 1) "구매정보 저장 옵션"에서 설정된 저장 위치 확인
    2) 파일명: jangbogo_날짜_시간.확장자 형식으로 검색
    3) Windows 탐색기 검색 기능 활용

 Q3: 자동 수집이 작동하지 않아요
 ───────────────────────────────────────────
 A: 1) 자동 수집 주기가 0보다 큰 값으로 설정되었는지 확인
    2) 체크박스가 선택되었는지 확인
    3) [자동수집] 버튼을 클릭하여 설정 저장했는지 확인
    4) 프로그램이 계속 실행 중인지 확인

 Q4: 여러 쇼핑몰을 동시에 수집할 수 있나요?
 ───────────────────────────────────────────
 A: 네, 가능합니다.
    - 여러 쇼핑몰의 체크박스를 선택한 후
    - [자동수집] 버튼을 클릭하면 순차적으로 수집됩니다.

 Q5: 과거 구매내역도 수집되나요?
 ───────────────────────────────────────────
 A: 네, 각 쇼핑몰에서 제공하는 범위 내에서 수집됩니다.
    - 일반적으로 최근 6개월~1년 데이터 수집 가능
    - 쇼핑몰마다 제공 범위가 다를 수 있습니다

 Q6: 저장 포맷은 무엇을 선택해야 하나요?
 ───────────────────────────────────────────
 A: 용도에 따라 선택하세요:
    - 엑셀로 보려면: CSV 또는 Excel
    - 다른 프로그램과 연동: JSON
    - 백업 목적: JSON (가장 완전한 데이터)

 Q7: 데이터는 어디에 저장되나요?
 ───────────────────────────────────────────
 A: 두 곳에 저장됩니다:
    1) 데이터베이스: db/jangbogo-dev.db (자동)
    2) 파일: 설정한 저장 위치 (수동/자동)

 Q8: 쇼핑몰 비밀번호를 변경했어요
 ───────────────────────────────────────────
 A: 해당 쇼핑몰의 [계정연결] 버튼을 다시 클릭하여
    새로운 비밀번호로 재연결하세요.

 Q9: 프로그램을 종료하면 자동 수집도 멈추나요?
 ───────────────────────────────────────────
 A: 네, 프로그램이 실행 중일 때만 자동 수집됩니다.
    - 24시간 자동 수집을 원하면 프로그램을 계속 실행
    - Windows 서비스로 등록하면 백그라운드 실행 가능
      (service 폴더의 README.md 참조)

 Q10: 저장된 데이터를 삭제하고 싶어요
 ───────────────────────────────────────────
 A: 1) 파일 삭제: 저장 위치의 파일을 직접 삭제
    2) 데이터베이스 초기화:
       - db/jangbogo-dev.db 파일 삭제
       - 프로그램 재시작 시 새로 생성됨
