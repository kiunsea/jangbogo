# JANGBOGO (장보고) v0.5.0 🛒

**온라인 쇼핑몰 구매내역 자동 수집·관리 프로그램**

---

## 환영합니다!

장보고는 여러 온라인 쇼핑몰의 **구매내역을 자동으로 수집**하여 **파일(JSON/CSV/Excel)**로 저장하는 프로그램입니다.

### 주요 기능

- 🛒 **다중 쇼핑몰 지원**: SSG, 오아시스, 하나로마트 등
- 📊 **자동 수집**: 설정한 주기마다 자동으로 수집
- 💾 **다양한 형식**: JSON, CSV, Excel로 저장
- 🔒 **로컬 저장**: 모든 데이터는 사용자 PC에 안전하게 보관
- 🌐 **웹 기반**: 브라우저에서 쉽게 관리

---

## 🚀 빠른 시작

### 1. 실행하기

**`Jangbago.bat` 파일을 더블클릭**

또는 명령 프롬프트에서:
```cmd
Jangbogo.bat
```

### 2. 브라우저 접속

자동으로 브라우저가 열립니다.

**접속 주소**: `http://127.0.0.1:8282`

### 3. 로그인

**기본 관리자 계정:**
```
아이디: admin_main
비밀번호: admin1234_main
```

⚠️ **보안을 위해 첫 로그인 후 계정을 변경하세요!**

변경 방법: `사용설명서.txt` 참조

## 📖 사용 방법

### 쇼핑몰 계정 연결

1. 로그인 후 하단 **"쇼핑몰 목록"** 섹션으로 이동
2. 연결하려는 쇼핑몰의 **[계정연결]** 버튼 클릭
3. 쇼핑몰 아이디/비밀번호 입력
4. 연결 완료!

### 구매내역 수집

1. 수집할 쇼핑몰의 **체크박스** 선택
2. **[자동수집]** 버튼 클릭
3. 수집 완료 메시지 확인

### 파일로 저장

1. 화면 중간 **"구매정보 저장 옵션"** 섹션에서 설정
2. **저장 위치**, **저장 포맷** 선택
3. **[지금저장]** 버튼 클릭

또는:
- **"구매내역 수집시 함께 저장"** 체크
- 수집할 때마다 자동 저장

## ❓ 문제 해결

### 프로그램이 실행되지 않아요

1. `Jangbogo.bat` 파일을 우클릭 → **"관리자 권한으로 실행"**
2. 로그 파일 확인: `logs\jangbogo.log`
3. 오류 메시지 확인 및 문의

### 브라우저가 안 열려요

수동으로 브라우저에서 접속:
```
http://127.0.0.1:8282
```

### 포트 충돌 오류

다른 프로그램이 8282 포트를 사용 중입니다.

**해결:**
```cmd
# 포트 사용 프로세스 확인
netstat -ano | findstr :8282

# 프로세스 종료
taskkill /F /PID [PID번호]
```
